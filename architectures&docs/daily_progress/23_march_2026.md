# 📅 Progress Update — 23 March 2026

## 🚀 Key Architectural Decision

Today, we made an important architectural improvement:

👉 **Separated platform setup from CI/CD pipeline logic**

### Previous Approach
- Jenkins setup + pipeline scripts + infra code were in the same repository
- This created tight coupling and reduced flexibility

### New Approach (Current)

```text
infra-terraform/        → VPC, EKS, IAM
platform-kubernetes/    → Jenkins setup (K8s manifests)
app / pipeline repos    → Jenkinsfile + application logic
